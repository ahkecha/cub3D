>  This project is inspired by the world-famous Wolfenstein 3D game, which was the first FPS ever. It will enable you to explore ray-casting. Your goal will be to make a dynamic view inside a maze, in which you‚Äôll have to find your way.

## Dependencies
> MinilibX (MacOS version)

## Build Project

* Make sure you have MinilibX installed or Use build the one in the ```mlx``` directory
* Go to mlx directory and ```make```
* After building MinilibX go back to the root directory of the project and build it

```sh
make
```
## Usage

```sh
./cub3d [map_file]; e.g ./cub3d maps/map2.cub

```
